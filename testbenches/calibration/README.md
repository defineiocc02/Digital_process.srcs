# 校准模块测试

## 测试平台

- `tb_gain_comp_check_lsb.sv` - 校准精度验证测试平台

## 测试内容

### 校准算法验证
验证校准算法的精度和可靠性：

1. **模拟物理误差**：模拟实际芯片中的基准误差
2. **运行校准**：等待 RTL 校准完成
3. **提取权重**：获取校准后的原始权重
4. **计算增益**：计算系统增益和偏移系数
5. **应用校正**：将校正系数应用到校准值
6. **误差分析**：计算最终的绝对残差（LSB 单位）

### 判定标准
- **通过标准**：残差 < 0.5 LSB
- **蒙特卡洛分析**：多次运行验证统计特性

## 关键特性

- ✅ **工业标准验证**：符合业界校准验证流程
- ✅ **蒙特卡洛分析**：支持多次随机运行
- ✅ **自动判定**：自动判断是否通过测试
- ✅ **统计分析**：记录误差分布

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|

| WEIGHT_WIDTH | 30 | 权重位数（Q18.12） |
| MC_RUNS | 5 | 蒙特卡洛运行次数 |
