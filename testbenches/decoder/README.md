# 译码器测试

## 测试平台

- `tb_flash_decoder.sv` - Flash 译码器功能测试平台

## 测试内容

### 译码器功能验证
验证 Flash 译码器的热码转二进制功能：

1. **正常热码测试**：验证理想热码输入的转换
2. **气泡错误测试**：验证气泡错误的纠正能力
3. **边界条件测试**：验证全 0、全 1 等边界情况

### 测试方法
- 施加不同的热码输入模式
- 验证输出的二进制码是否正确
- 检查气泡错误是否被正确纠正

## 关键特性

- ✅ **全覆盖测试**：测试所有可能的输入模式
- ✅ **气泡注入**：主动注入气泡错误验证纠错能力
- ✅ **自动验证**：自动比对期望值和实际值
- ✅ **详细日志**：输出详细的测试结果

## 测试用例

### 1. 正常热码测试
```
输入 (thermo) | 期望输出 (binary) | 说明
--------------|-------------------|------
3'b000        | 2'b00            | 理想 0
3'b001        | 2'b01            | 理想 1
3'b011        | 2'b10            | 理想 2
3'b111        | 2'b11            | 理想 3
```

### 2. 气泡错误测试
```
输入 (thermo) | 期望输出 (binary) | 错误类型
--------------|-------------------|----------
3'b101        | 2'b10            | 气泡（中间有 0）
3'b010        | 2'b01            | 气泡（中间有 1）
3'b100        | 2'b01            | 气泡（高位异常）
```

## 预期结果

- 所有正常热码输入转换正确
- 所有气泡错误被正确纠正
- 边界条件处理正确
