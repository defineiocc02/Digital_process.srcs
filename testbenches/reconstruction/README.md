# 重构模块测试

## 测试平台

- `tb_sar_recon.sv` - 重构模块功能测试平台

## 测试内容

### 重构功能验证
验证重构模块的功能和性能：

1. **线性度测试**：验证输入电压与输出码的线性关系
2. **权重更新灵敏度测试**：验证权重变化对输出的影响
3. **流水线吞吐测试**：验证连续输入时的处理能力

### 测试方法
- 使用 `generate_ideal_bits(vin)` 将归一化电压（-1~+1）转换为 20-bit 理想码
- 使用 `force_ideal_weights()` 灌入理想权重 W[i] ∝ 2^(i+4)
- 验证输出与理论值的偏差

## 关键特性

- ✅ **理想权重注入**：支持手动设置权重值
- ✅ **自动验证**：自动比对理论值和实际值
- ✅ **流水线测试**：验证连续输入场景
- ✅ **误差分析**：记录并分析量化误差

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| CAP_NUM | 20 | 电容总位数 |
| WEIGHT_WIDTH | 30 | 权重位数 |
| OUTPUT_WIDTH | 16 | 输出位数 |
| FRAC_BITS | 8 | 权重小数位数 |

## 测试用例

### Test 1: 线性度测试
- 扫描输入电压范围（-1V 到 +1V）
- 验证输出码的线性度
- 判定标准：误差 < ±1 LSB

### Test 2: 权重更新测试
- 动态修改权重值
- 验证输出响应
- 确认权重更新机制正常工作

### Test 3: 流水线吞吐测试
- 连续输入多个样本
- 验证输出数据完整性
- 测试背压处理能力
