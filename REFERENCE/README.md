## 📝 使用说明

### 添加新文献

1. 将 PDF 或其他格式的文献文件放入本文件夹
2. 更新本文档的"当前文献"章节
3. 在相应分类中添加记录

### 引用文献

在项目的其他文档中引用本文件夹的文献时，请使用标准引用格式：

```
作者，"标题，" 出版物，年份.
```

### 查找文献

- 按文件名搜索：文献文件名通常包含作者和年份信息
- 按主题搜索：参考上方的分类表格
- 按相关性搜索：查看"与本项目的相关性"说明
