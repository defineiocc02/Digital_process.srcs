# 顶层模块

## 功能描述

FPGA 顶层包装模块，集成校准控制器和虚拟 ADC 物理模型，用于 FPGA 板级验证。

## 核心文件

- `fpga_top_wrapper.sv` - FPGA 顶层包装器

## 系统架构

```
┌───────────────────────────────────────────────┐

├───────────────────────────────────────────────┤
│                                               │

```

## 工作原理

### 校准模式
1. **启动**：通过 start_sw 开关启动校准
2. **执行**：校准控制器驱动虚拟 ADC 物理模型
3. **完成**：校准完成后，done_led 指示灯亮起
4. **权重输出**：通过 w_wr_* 接口输出校准权重

## 接口说明

### 输入信号
- `clk` - 板载时钟（如 100MHz 或 50MHz）
- `rst_n_btn` - 复位按键（低电平有效）
- `start_sw` - 启动开关

### 输出信号
- `done_led` - 校准完成指示灯
- `w_wr_en` - 权重写使能（调试用）
- `w_wr_addr` - 权重写地址（调试用）
- `w_wr_data` - 权重写数据（调试用）

## 关键特性

- ✅ **独立校准**：无需外部 ADC 即可验证校准算法
- ✅ **调试友好**：关键信号添加 mark_debug 属性
- ✅ **板级验证**：直接映射到 FPGA 引脚和 LED
- ✅ **ILA 支持**：预留 ILA 调试核接口

## 约束文件

- `constraints/sar_calib_fpga.xdc` - FPGA 约束文件
  - 时钟约束：20ns (50MHz)
  - 引脚分配：clk, rst_n_btn, start_sw, done_led
  - ILA 调试约束

### 3. 操作流程

2. 拨动 start_sw 开关启动校准
3. 等待 done_led 指示灯亮起
4. 通过 ILA 观察校准过程和权重值

## 注意事项

1. **复位消抖**：建议对 rst_n_btn 添加消抖电路
2. **时钟频率**：确保时钟频率满足时序约束
3. **ILA 深度**：根据校准时间调整 ILA 捕获深度
4. **权重保存**：校准完成后保存权重用于重构

## 调试技巧

- 使用 ILA 观察 dac_p_force/dac_n_force 的变化
- 监控 comp_out 信号验证比较结果
- 捕获 w_wr_data 验证权重计算结果
- 观察状态机转换过程
