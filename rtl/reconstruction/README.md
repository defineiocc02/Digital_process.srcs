# 重构模块

## 功能描述

实现 SAR ADC 的数字重构引擎，使用校准后的权重对 SAR 原始数据进行加权求和，转换为高精度的 16 位有符号数字码。

## 核心文件

- `sar_reconstruction.sv` - 重构引擎主模块

## 工作原理

### 重构流程
1. **权重存储**：使用本地 RAM 存储各比特权重，支持动态更新
2. **部分累加**：将 20 个输入分为 4 组，每组 5 个输入，计算各组部分和
3. **全局累加**：将 4 组部分和相加得到总和
4. **缩放与饱和**：
   - 除以 2（因为输入范围是 2*Vref）
   - 加 0.5 LSB 进行舍入
   - 右移 FRAC_BITS 位进行缩放
   - 饱和处理，确保输出在 16 位有符号范围内

## 关键特性

- ✅ **流水线设计**：分为部分累加 (Stage 1) 和全局累加 (Stage 2)
- ✅ **40 位动态范围**：防止中间结果溢出
- ✅ **0.5 LSB 偏移补偿**：提高输出精度
- ✅ **动态权重更新**：支持实时校准写入

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| CAP_NUM | 20 | 电容总位数 |
| WEIGHT_WIDTH | 30 | 权重存储位数（有符号） |
| OUTPUT_WIDTH | 16 | 输出数据位数 |
| FRAC_BITS | 8 | 权重小数部分位数（Q22.8 格式） |
| TOTAL_SHIFT | FRAC_BITS | 总移位数 |
| GROUP_SIZE | 5 | 部分累加分组大小 |

## 接口说明

### 输入信号
- `clk` - 系统时钟
- `rst_n` - 异步复位（低电平有效）
- `data_valid_in` - SAR 转换完成标志
- `raw_bits` - 原始 SAR 数据（20-bit）
- `w_wr_en` - 权重写使能
- `w_wr_addr` - 权重写地址
- `w_wr_data` - 校准后的权重（30-bit 有符号）

### 输出信号
- `adc_dout` - 重构后的 ADC 输出（16-bit 有符号）
- `data_valid_out` - 输出数据有效标志

## 设计注意事项

1. **符号扩展**：使用 signed'() 进行符号扩展，确保计算正确性
2. **舍入处理**：加 0.5 LSB 进行四舍五入，减少量化误差
3. **饱和逻辑**：防止数值溢出导致符号翻转

## 性能指标

- **延迟**：2 个时钟周期（两级流水线）
- **吞吐率**：每时钟周期一个样本
- **精度**：16-bit 有符号输出
