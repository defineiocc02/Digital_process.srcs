# 顶层系统测试

## 测试平台

- `tb_sar_adc_top.sv` - 完整系统测试平台

## 测试内容

### 完整系统验证
测试整个 SAR ADC 数字处理系统的功能，包括：
- 校准控制器
- 虚拟 ADC 物理模型
- SAR 逻辑控制器
- 重构引擎

### 测试流程
1. **校准阶段**：运行校准算法，测量并存储权重
2. **验证阶段**：使用校准后的权重进行重构
3. **性能评估**：检查线性度、INL/DNL 等指标

## 关键特性

- ✅ **端到端测试**：验证完整信号链
- ✅ **自动验证**：自动检查校准和重构结果
- ✅ **性能统计**：记录 INL/DNL 数据
- ✅ **可视化**：生成波形用于调试

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| CAP_NUM | 20 | 电容总位数 |
| WEIGHT_WIDTH | 30 | 权重位数 |
| OUTPUT_WIDTH | 16 | 输出位数 |
| CLK_PERIOD | 200 | 时钟周期（5MHz） |

## 预期结果

- 校准完成后，权重应接近理论值
- 重构输出应与输入电压成线性关系
- INL < 1.0 LSB（全范围）

## 调试技巧

- 观察校准过程中的权重变化
- 检查重构输出的线性度
- 对比理想值和实际值
