# SAR 逻辑控制模块

## 功能描述

实现 SAR ADC 的逐次逼近控制逻辑，控制 SAR 转换过程的时序和状态机。

## 核心文件

- `sar_adc_controller.sv` - SAR 控制器主模块

## 工作原理

### SAR 转换流程
1. **启动转换**：接收 start 信号，开始 SAR 转换
2. **逐次逼近**：从高位到低位依次试探
3. **比较判断**：根据比较器输出决定保留或丢弃当前位
4. **转换完成**：所有位试探完成后，输出结果并拉高 eoc 信号

## 关键特性

- ✅ **逐次逼近算法**：二分搜索策略
- ✅ **可配置位数**：支持 CAP_NUM 参数配置
- ✅ **低延迟设计**：优化的状态机设计
- ✅ **结果验证**：输出有效标志

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| CAP_NUM | 20 | SAR 转换位数 |

## 接口说明

### 输入信号
- `clk` - 系统时钟
- `rst_n` - 异步复位（低电平有效）
- `start` - 转换启动信号
- `comp_out` - 比较器输出

### 输出信号
- `eoc` - 转换结束标志
- `result_out` - SAR 转换结果
- `result_valid` - 结果有效标志
- `dac_p_force` - P 路 DAC 控制信号

## 设计注意事项

1. **时序约束**：确保比较器有足够稳定时间
2. **功耗优化**：仅在转换期间激活相关逻辑
3. **抗干扰**：考虑添加数字滤波

## 测试建议

- 验证所有输入码的转换正确性
- 测试边界条件（全 0、全 1）
- 验证时序裕量
