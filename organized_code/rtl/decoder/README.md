# Flash 译码器模块

## 功能描述

实现热码（Thermometer Code）到二进制码的转换，并集成加法器功能，用于高速 ADC 数据的译码处理。

## 核心文件

- `flash_decoder_adder.sv` - Flash 译码器 + 加法器模块

## 工作原理

### 译码流程
1. **热码输入**：接收来自比较器阵列的热码信号
2. **气泡纠错**：检测并修正热码中的气泡错误
3. **二进制转换**：将热码转换为二进制码
4. **加法运算**：对转换后的数据进行加法处理

### 气泡错误处理
- **类型 1**：中间有 0（如 101）→ 修正为 111
- **类型 2**：中间有 1（如 010）→ 修正为 011
- **类型 3**：高位异常（如 100）→ 修正为 011

## 关键特性

- ✅ **高速译码**：组合逻辑实现，无时钟延迟
- ✅ **气泡容错**：自动检测并修正常见气泡错误
- ✅ **集成加法**：译码 + 加法一体化设计
- ✅ **可配置位宽**：支持参数化配置

## 参数配置

| 参数 | 默认值 | 说明 |
|------|--------|------|
| THERMO_WIDTH | 可配置 | 热码输入位宽 |
| BINARY_WIDTH | 可配置 | 二进制输出位宽 |

## 接口说明

### 输入信号
- `thermo_in` - 热码输入信号
- `addend_a` - 加法器输入 A（可选）
- `addend_b` - 加法器输入 B（可选）

### 输出信号
- `binary_out` - 二进制输出
- `sum_out` - 加法结果（可选）

## 设计注意事项

1. **位宽匹配**：确保热码位宽与二进制位宽匹配（2^N - 1）
2. **错误检测**：气泡纠错逻辑的覆盖范围
3. **时序优化**：关键路径优化，提高工作频率

## 应用场景

- Flash ADC 数据译码
- 高速比较器阵列输出处理
- 温度计码转换
